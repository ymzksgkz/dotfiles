## Install
```bash
make install
```

## How to use
```bash
export NVIM_CUSTOM_ENV_NAME='your_custom_env_name'

alias vim="NVIM_APPNAME=$NVIM_CUSTOM_ENV_NAME nvim"
nvim your_file_name
```

## Uninstall
```bash
make uninstall
```

## Reinstall
```bash
make reinstall
```

## Setup Github Copilot
```bash
# nvim command
:Copilot setup
```
